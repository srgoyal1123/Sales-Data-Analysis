# Sales-Data-Analysis
An Excel-based sales dashboard that analyzes and visualizes business data using Pivot Table and charts. 
# Objective 
The main objective of this project is to analyze sales data and present it in a clear and interactive dashboard to support better decesion-making.
#Dataset Details.
The dataset includes ORDER ID,ORERD DATE,REGION,SALESPERSON,PRODUCT,QUANTITY,UNIT PRICE,TOTAL SALES.
# Dashboard Insights
The dashboard provides key business insights such as:
Total Sales and Total orders
Top performing region
Top salesperson 
Best selling product 
Total revenue and average sales 
# Features
Data cleaning and organization 
Pivot tables for analysis 
Interactive filters 
Charts for better visualization 
KPI metrics 
# Project Overview . 
This project is an excel based sales dashboard designed to analyse and visualize business data. It helps in understanding sales performance across different regions , products , and salesperson using interactive charts and pivot tables.
